# 📸 Instagram Clone

A full-stack social media web application inspired by Instagram, built using **Django**. Users can create accounts, share posts, interact through likes, comments, follows, stories, notifications, and one-to-one chat.

---

## 🚀 Features

- User Registration, Login & Logout
- Secure User Authentication
- User Profile Management
- Edit Profile
- Create Posts (Image/Video)
- Edit Posts
- Delete Posts
- Home Feed
- Like & Unlike Posts
- Comment on Posts
- Reply to Comments
- Delete Comments
- Follow & Unfollow Users
- Stories
- Notifications
- Search Users
- One-to-One Chat
- View Single Post

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Harshitha625/Instagram_clone.git
```

### Navigate to the project

```bash
cd Instagram_clone
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/macOS**

```bash
source venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Apply migrations

```bash
python manage.py migrate
```

### Run the development server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
